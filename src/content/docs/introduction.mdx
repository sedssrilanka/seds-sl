# Welcome to SEDS Sri Lanka Documentation

Welcome to the official documentation for the **SEDS Sri Lanka** website!
This platform serves as the central hub for our divisions, chapters, projects, and e-commerce shop, all powered by **Next.js** and **Payload CMS**.

![SEDS Sri Lanka Homepage](https://placehold.co/800x400/0f172a/ffffff?text=SEDS+Sri+Lanka+Homepage)

## What is this platform?

This website is a full-stack Next.js 15 application utilizing the new App Router and integrated with Payload CMS 3.0 to manage content seamlessly. It uses Tailwind CSS for styling and shadcn/ui for accessible components.
